* Measuring cable frequency response
** Using FuzzMeasure
   - Output 3 on UltraLite is output 5 in FuzzMeasure
   - Watch the gains so that they don't distort:
     - Output gain in FuzzMeasure: -5 dB
     - Input gain (trim) in CueMix: 0 dB
